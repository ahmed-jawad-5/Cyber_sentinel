If the other laptop is not on the same subnet (e.g., 10.96.166.x), they won’t be able to reach each other.

2️⃣ Firewall / security software

Windows / Linux firewalls often block incoming connections.

On Linux, check with:

sudo ufw status


If active, allow your port:

sudo ufw allow 9000/tcp
sudo ufw allow 9000/udp  # if needed


On Windows, make sure inbound rules allow Python / port 9000.

3️⃣ Receiver binding only to localhost
